# 监控大盘

用友云开发者中心的监控大盘可以让用户查看应用的**页面访问量曲线**、**用户访问量曲线**、**全国地区访问量**等其他的信息。

点击菜单栏左侧的【监控大盘】，进入【监控大盘】页面，如图1所示。
